MINESWEEPER

PETSTEPS

⇀ Mobile fitness app created using Swift in Xcode

⇀ Mobile game format where players take care of pets and earn coins via exercise

⇀ Coins can be used to purchase food for pets in the in-game shop and purchase more pets

⇀ Augmented reality option allows players to view their pets in a real world setting

⇀ Players can message an in-app AI chatbot for fitness tips and other pressing issues

WEATHER APP

⇀ Recreates the default weather app on smartphones using Swift on Xcode that accurately retrieves the current weather forecast for locations within the United States

⇀ Users can save locations for easier access on the home screen

⇀ Users can search for specific locations by typing the requested city and state into the provided searchbox

⇀ Users can also save past searches for ease of access
